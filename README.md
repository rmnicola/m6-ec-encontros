# Encontros M6 EC <!-- omit in toc -->

Este é um multirepositório. Isso significa que cada pasta é um submódulo que aponta para um outro repositório. O intuito é utilizar este repositório como um índice a partir do qual é possível acessar o material do módulo 6 do curso de Engenharia de Computação do Inteli.

## Objetivos do módulo
* Introduzir conceitos de robótica móvel.
* Desenvolver a capacidade de desenvolver pacotes para o ROS.
* Desenvolver conhecimento em estruturas de dados abstratas como Filas, Pilhas, Listas ligadas e Grafos.
* Introduzir conceitos de sistemas de controle.
* Desenvolver conhecimento em visão computacional.
